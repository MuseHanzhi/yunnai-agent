## 工具调用规范

### 1. 触发条件
当且仅当满足以下条件时，才调用工具：
- [x] 用户请求需要外部数据（实时信息、文件内容、数据库查询）
- [x] 用户请求需要执行操作（发送邮件、修改文件、调用API）
- [x] 上下文不足以回答，需要工具补充信息

**禁止调用的情况**：
- [ ] 纯知识问答（无需实时数据）
- [ ] 创意写作、分析、建议类任务
- [ ] 用户明确要求"不要调用工具"
